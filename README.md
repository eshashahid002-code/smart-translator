# 🌍 AI Language Translator

A smart web-based language translator built using HTML, CSS, and JavaScript. This application allows users to translate text between multiple languages, convert emojis into text, extract text from images (OCR), use voice input, listen to translated text, copy results, and download translations.

## 🚀 Features

* 🌐 Multi-language text translation
* 🔄 Language swap functionality
* 🎤 Voice-to-text input
* 🔊 Text-to-speech output
* 😀 Emoji to text conversion
* 📷 OCR (Image to Text) using Tesseract.js
* 📋 Copy translated text to clipboard
* 🗑️ Clear input and output fields
* 📥 Download translation as a text file
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* MyMemory Translation API
* Tesseract.js OCR
* Web Speech API

## ⚙️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
4. Start translating text instantly.
